Test API
=================
The test API coding exercise.

Author
--------
Lincoln Li

Base URL
--------
- /test/1.0

Endpoints
--------
- /tasks/validateBrackets (GET): Checks if brackets in a string are balanced
- /todo (POST): Create a to do item
- /todo/{id} (GET): Retrieve a specific item by id
- /todo/{id} (PATCH): Modify an item
